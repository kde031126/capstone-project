# Team 39 NexStep – 음성인식 AI 및 음소 분석 기반, 개인화된 아동 발음 오류 교정 및 언어 놀이 앱

---

## 📌 프로젝트 개요
- **주제**: 음성인식 AI 및 음소 분석 기반, 개인화된 아동 발음 오류 교정 및 언어 놀이 애플리케이션
- **대상**: 발음 지연 또는 언어 발달이 느린 6~12세 아동
- **목표**  
  1) 아이는 놀이처럼 즐겁게 말하기 연습  
  2) 보호자는 한눈에 학습 진행 상황 확인  

- **핵심 기능**  
  1) 음성인식 AI와 음소 분석을 통한 아동 발음 교정  
  2) 아동 맞춤형 학습 단어 재구성  
  3) 보호자용 학습 요약 리포트 제공

---

## 🧩 Pain Point
- **일상 속 지속적 연습의 어려움**: 아동 혼자 훈련 유지 어려움 / 보호자의 시간 부족  
- **개인 수준 맞춤 프로그램 설계의 어려움**: 아동별 발달 속도와 발음 오류 유형 상이  
- **발음 정확도 확인의 한계**: 아동 스스로 발음 교정 어려움  
- **진행 상황 관리 비효율성**: 보호자가 학습 진행/결과 확인 어려움  

---

## 🏗️ 솔루션 컨셉
1. **진단 테스트** – 아이의 현재 발음 수준을 빠르게 파악
2. **발음 오류 분석** – Wav2Vec 2.0 → 아이 발음 인식
                       g2pK → 표준 발음과 비교
                       음소 단위로 오류 자동 탐지
3. **개인 맞춤 훈련** – 아이에게 필요한 단어와 활동만 추천
4. **학습 요약 리포트** - OpenAI GPT-5로 학습 결과를 쉽게 정리
---

## 🛠️ 기술 스택
- **Frontend**: React.js (https://ko.legacy.reactjs.org/)  
- **Backend**: Node.js (https://nodejs.org/ko/)  
- **Database**: MySQL (https://www.mysql.com/)  
- **AI/ML**:
  - **Wav2Vec 2.0**: 음성 → 텍스트 변환 (https://ai.meta.com/research/impact/wav2vec/)  
  - **g2pK**: 단어 표준 발음 변환 (https://github.com/Kyubyong/g2pK)  
  - **OpenAI GPT-5**: 개인 맞춤형 프로그램 생성, 진행 상황 요약 (https://platform.openai.com/docs/overview)

## 🚀 기대 효과
- 아동의 **발음 및 표현 능력 향상**  
- 집에서도 **지속적 학습 가능** → 치료 연속성 강화  
- **보호자 관리 부담 감소**, 학습 데이터 자동 기록 및 시각화
