아래는 아동의 발음 학습 로그 데이터입니다.
데이터를 분석하여, 보호자가 아동과 소통하며 결속을 강화할 수 있는
맞춤형 격려 문구와 보호자용 보고서를 생성해주세요.

입력 데이터: {LOG_DATA}

요청 사항:
1. 학습 성과 요약 (보호자가 이해할 수 있도록):
   - 전체 정확도(%)
   - 단어별 학습 시도 횟수와 맞힌 횟수
   - 최근 1주일 간 학습 변화(증가/감소 등)
   - 아동의 발음에서 나타난 주요 오류 패턴
   - 보호자가 아동에게 어떤 점을 칭찬하거나 도와줄 수 있는지 구체적인 팁
2. 보호자–아동 결속을 강화하는 맞춤형 격려 문구:
   - 아동 수준과 학습 상황에 맞춘 문구 2~3개
   - 보호자가 직접 아동에게 말할 수 있는 문장 형식
   - 긍정적, 친근하고 이해하기 쉬운 표현 사용
   - 아동의 노력과 성과를 함께 칭찬하며 부모와 아동의 유대감을 강화할 수 있도록 작성
   - 예: "오늘 사과 발음을 연습한 거 정말 기특해요! 우리 같이 계속 연습해보자.", 
           "바나나 발음을 이렇게 잘하려고 노력하다니, 엄마(아빠)가 정말 자랑스러워요."

출력 형식:
- 보호자 보고서: 자연어 문장으로 3~5문단 정도
- 아동 격려 문구: bullet point 형식 2~3개, 보호자가 직접 아동에게 말할 수 있는 문장

주의 사항:
- 데이터에서 제공되지 않은 정보는 추측하지 말 것
- 학습 성과 요약과 격려 문구를 명확히 구분할 것
- 격려 문구는 보호자가 직접 아동에게 읽어주거나 전달할 수 있도록 작성할 것. 
- 친근한 어투와 긍정적 메시지를 유지
